# RAG AI Assistant

A Retrieval-Augmented Generation (RAG) application using ChromaDB, Ollama, and FastAPI.

## 🚀 Features
- Document upload (PDF/TXT)
- RAG-based question answering
- Multiple LLM models via Ollama
- Text-to-Speech (TTS) support
- Real-time chat interface

## 🛠️ Tech Stack
- **Backend**: FastAPI, ChromaDB, Ollama
- **Frontend**: HTML, TailwindCSS, Alpine.js
- **ML**: Sentence Transformers, Ollama (phi3:mini)
- **TTS**: Piper

## 📋 Prerequisites
- Python 3.10+
- Ollama installed and running
- Conda (recommended)
